Before Starting Tests, Please Ensure the Following Steps
  • Click "Download Sample File", fill in test data, then start the test.
  • Start both ADB and Appium servers.
  • Enable USB Debugging: Settings → About Phone → Build Number (tap 7×) → Developer Options → USB Debugging.
  • USB: Enable "Install via USB".
  • Wireless: Enable "Wireless Debugging" — note IP, Port & Pairing Code.
  • Ensure device & computer share the same VPN (wireless only).
